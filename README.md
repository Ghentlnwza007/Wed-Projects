# MAISON - Premium Lifestyle Wear

Web Application for MAISON, a premium fashion brand.

## How to View the Website

### Option 1: View Online (GitHub Pages)

The website is hosted on GitHub Pages. You can visit it at:
**[https://Ghentlnwza007.github.io/Wed-Projects/](https://Ghentlnwza007.github.io/Wed-Projects/)**

_Note: If you just pushed new code, it might take 1-2 minutes to update._

### Option 2: Run Locally (Recommended for Development)

To view the site on your computer without internet or waiting for deployment:

1. Open a terminal (Command Prompt or PowerShell) in this folder.
2. Run the following command:
   ```powershell
   python -m http.server
   ```
3. Open your browser and go to: [http://localhost:8000](http://localhost:8000)
